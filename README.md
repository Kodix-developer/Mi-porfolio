# Web personal de servicios

Mi web de presentación como desarrollador web para negocios locales y autónomos.

## Estructura
```
Mi porfolio/
├── index.html        → la página (7 secciones)
├── css/styles.css    → todos los estilos (colores en :root)
├── js/main.js        → menú móvil, año, validación del formulario
├── img/              → imágenes y favicon
└── CONTEXTO.md       → plan de negocio y hoja de ruta
```

## Cómo verla
Abre `index.html` haciendo doble clic, o usa "Live Server" en VS Code
para que se recargue sola al guardar cambios.
